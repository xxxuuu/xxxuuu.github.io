乘法逆元

2019-05-25

逆元就是用来解决大数除法取模计算的

取模规则

a+b(modp)=(a mod p+b mod p) mod pa + b \pmod{p} = (a \bmod{p} + b \bmod{p}) \bmod{p} a+b(modp)=(amodp+bmodp)modp

a−b(modp)=(a mod p−b mod p) mod pa - b \pmod{p} = (a \bmod{p} - b \bmod{p}) \bmod{p} a−b(modp)=(amodp−bmodp)modp

a×b(modp)=(a mod p×b mod p) mod pa \times b \pmod{p} = (a \bmod{p} \times b \bmod{p}) \bmod{p} a×b(modp)=(amodp×bmodp)modp

ab(modp)≠(a mod pb mod p) mod p\frac{a}{b} \pmod{p} \neq (\frac{a \bmod{p}}{b \bmod{p}}) \bmod{p} ba​(modp)​=(bmodpamodp​)modp

逆元定义

根据取模规则可以发现，对于

ab(modp)\frac{a}{b} \pmod{p} ba​(modp)

当 bbb 非常大时，因为无法拆分取模，结果难以计算，这时就需要逆元。逆元本质就是把除法取模转换为乘法以便于拆分计算

我们知道：

若a×b=1，则 b 为 a 倒数，即：b=1a若 a \times b = 1，则\ b\ 为\ a\ 倒数，即：b = \frac{1}{a} 若a×b=1，则 b 为 a 倒数，即：b=a1​

但当：

a×b≡1(modp) 时，b 不一定为 1a  (因为有取模)a \times b \equiv 1 \pmod{p} \ 时，b\ 不一定为\ \frac{1}{a} \ \ (因为有取模) a×b≡1(modp) 时，b 不一定为 a1​  (因为有取模)

若 aaa 与 ppp 互质。我们就把 bbb 称为 aaa 关于 ppp 的逆元。记作 a−1a^{-1}a−1。

例如：

2×3(mod5)≡12 \times 3 \pmod{5} \equiv 1 2×3(mod5)≡1

那么 333 就是 222 关于 555 的逆元

所以除以一个数再取模等于乘以其逆元再取模，所以逆元又称数论倒数。即：

ab ⁣ ⁣ ⁣ ⁣(modp)=a×b−1 ⁣ ⁣ ⁣(modp)\frac{a}{b} \!\!\!\! \pmod{p} = a \times b^{-1} \!\!\! \pmod{p} ba​(modp)=a×b−1(modp)

求逆元

费马小定理 + 快速幂

对于小于质数 ppp 的所有正整数 aaa，都有 ap≡a ⁣(modp)a^p \equiv a \!\pmod{p}ap≡a(modp)，ap−1≡1 ⁣(modp)a^{p-1} \equiv 1 \! \pmod pap−1≡1(modp)

推得：

由 a×a−1≡1(modp)由 \ a \times a^{-1} \equiv 1 \pmod{p} 由 a×a−1≡1(modp)

ap−1≡1 ⁣(modp)a^{p-1} \equiv 1 \! \pmod p ap−1≡1(modp)

得 a×a−1≡ap−1(modp)得 \ a \times a^{-1} \equiv a^{p-1} \pmod{p} 得 a×a−1≡ap−1(modp)

a−1≡ap−2(modp)a^{-1} \equiv a^{p-2} \pmod{p} a−1≡ap−2(modp)

可以通过快速幂求出 ap−2(modp)a^{p-2} \pmod{p}ap−2(modp) 的值，就是逆元

ll fpm(ll x, ll power, ll mod) {
    x %= mod;
    ll ans = 1;
    for (; power; power >>= 1, (x *= x) %= mod)
        if(power & 1) (ans *= x) %= mod;
    return ans;
}
int main() {
    ll x = fpm(a, p - 2, p); //x为a在mod p意义下的逆元
}

线性递推

这个适合用于求一连串数字的逆元。

设

p=k×a+r, (1<r<a<p)   也就是 k 是 pa 的商, r 是余数p = k \times a + r,\ (1 < r < a < p)\ \ \ 也就是\ k \ 是\ \frac{p}{a}\ 的商,\ r\ 是余数 p=k×a+r, (1<r<a<p)   也就是 k 是 ap​ 的商, r 是余数

变形得:

k×a+r≡0(modp)k \times a + r \equiv 0 \pmod{p} k×a+r≡0(modp)

两边乘以 a−1a^{-1}a−1 和 r−1r^{-1}r−1 可得：

k×r−1+a−1≡0(modp)k \times r^{-1} + a^{-1} \equiv 0 \pmod{p} k×r−1+a−1≡0(modp)

a−1≡−k×r−1(modp)a^{-1} \equiv {-k} \times r^{-1} \pmod{p} a−1≡−k×r−1(modp)

a−1≡−pa×(p mod a)−1(modp)a^{-1} \equiv -\frac{p}{a} \times (p \bmod{a})^{-1} \pmod{p} a−1≡−ap​×(pmoda)−1(modp)

inv[1] = 1;
for(int a = 2; a < p; ++ i)
    inv[a] = (p - p / a) * inv[p % a] % p;  // 加个p是为了保证非负
