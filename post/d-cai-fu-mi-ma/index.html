基础数论题，费马小定理+逆元

题意

求有多少个 n(1≤n≤x)n (1 \leq n \leq x)n(1≤n≤x) 满足 nan≡b(modp)na^n \equiv b \pmod{p}nan≡b(modp) ，其中 ppp 为质数。

输入给定 a,b,p,x(2≤p≤106,1≤x≤1012,1≤a,b<p)a, b, p, x(2 \leq p \leq 10^6, 1 \leq x \leq 10^{12}, 1 \leq a,b < p)a,b,p,x(2≤p≤106,1≤x≤1012,1≤a,b<p)

题解

首先，由费马小定理可以推出以下公式：

an≡an mod p−1(modp)a^n \equiv a^{n \bmod{p-1}} \pmod{p} an≡anmodp−1(modp)

  1. 当 n<p−1n < p-1n<p−1 时，an=an mod p−1a^n = a^{n \bmod{p-1}}an=anmodp−1

  2. 当 n=k(p−1)n = k(p-1)n=k(p−1) 时，n mod p−1=0n \bmod{p-1} = 0nmodp−1=0，则 an mod p−1=a0=1a^{n \bmod{p-1}} = a^0 = 1anmodp−1=a0=1

  3. 剩下情况可以用数学归纳法推出来

那么设：

n=k(p−1)+t(0≤t≤p−1)n = k(p-1) + t \quad (0 \leq t \leq p-1) n=k(p−1)+t(0≤t≤p−1)

代入原式：

nan≡b(modp)(k(p−1)+t)ak(p−1)+t≡b(modp)k(p−1)+t≡ba−t(modp)k≡(ba−t−t)(p−1)−1(modp)na^n \equiv b \pmod{p} \\ (k(p-1)+t)a^{k(p-1)+t} \equiv b \pmod{p} \\ k(p-1) + t \equiv ba^{-t} \pmod{p} \\ k \equiv (ba^{-t} -t)(p-1)^{-1} \pmod{p} nan≡b(modp)(k(p−1)+t)ak(p−1)+t≡b(modp)k(p−1)+t≡ba−t(modp)k≡(ba−t−t)(p−1)−1(modp)

由于 t≤p−1,p∈[2,106]t \leq p-1, p \in [2, 10^6]t≤p−1,p∈[2,106]，故可以通过枚举 ttt 来计算 kkk

设求出来的值为 k0∈[0,p)k_0 \in [0,p)k0​∈[0,p) 且 k≡k0(modp)k \equiv k_0 \pmod{p}k≡k0​(modp)

那么 k=m⋅p+k0k = m \cdot p + k_0k=m⋅p+k0​

最后我们只需要算出对于每个 ttt，有多少个 m≥0m \geq 0m≥0 使得原式成立

n=(mp+k0)(p−1)+t=mp(p−1)+k0(p−1)+t≤xm=x−k0(p−1)−tp(p−1)n = (mp+k_0)(p-1) + t = mp(p-1) + k_0(p-1) + t \leq x \\ m = \frac{x-k_0(p-1)-t}{p(p-1)} n=(mp+k0​)(p−1)+t=mp(p−1)+k0​(p−1)+t≤xm=p(p−1)x−k0​(p−1)−t​

代码实现

#include <iostream>
typedef long long ll;
using namespace std;

ll fpm(ll a, ll b, ll mod) {
    ll res = 1, temp = a;
    for (; b; b = b >> 1) {
        if (b & 1) {
            res = res * temp % mod;
        }
        temp = temp * temp % mod;
    }
    return res;
}


ll a,b,p,x;
int main() {
    cin >> a >> b >> p >> x;

    // 求p-1在模p意义下的逆元
    ll invP = fpm(p-1, p-2, p);

    ll ans = 0;
    // 枚举t来计算k
    for(int t = 0; t < p-1; t++) {
        ll invA = fpm(fpm(a,t,p), p-2, p);
        ll k = (b * invA - t) * invP % p;
        k = (k + p) % p;
        ll d = x - k * (p - 1) - t;
        if(d >= 0) {
            ans += d / (p * (p - 1)) + 1;
        }
    }

    cout << ans << endl;
    return 0;
}
